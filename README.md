# Track It!

Simpler Workout-Tracker für mich :D

## Features

- **Workouts erstellen** mit beliebigen Übungen und Sätzen
- **Satz-Modi:** Normal oder Links/Rechts
- **Finisher:** AMRAP, EMOM oder Normal
- **History:** Alle Workouts chronologisch sortiert
- **Bearbeiten & Löschen** von gespeicherten Workouts
- **Markdown-Export:** Workout als formatierten Text kopieren
- **Auto-Draft:** Entwurf wird automatisch gespeichert und bei Neustart wiederhergestellt
- **Export/Import:** Alle Daten als JSON sichern und wiederherstellen
- **Offline-First:** Funktioniert komplett ohne Internet (PWA)

Wer es findet, have fun :)
